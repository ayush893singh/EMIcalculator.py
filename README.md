# EMI Calculator

A simple Python project that calculates the monthly EMI (Equated Monthly Installment) for a loan based on loan amount, interest rate, and loan period.

---

Features

- Calculate monthly EMI instantly
- Supports any loan amount
- Supports custom interest rates
- Supports custom loan duration
- Beginner-friendly Python project

---

Technologies Used

- Python

---

This project uses:

- Variables
- User Input

- Arithmetic Operators
- Output Formatting

---

Installation

1. Install Python on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.

---

How to Run

python main.py

---

How It Works

1. User enters the loan amount.
2. User enters the annual interest rate.
3. User enters the loan period in years.
4. The program calculates the monthly EMI.
5. The EMI amount is displayed.

---

Formula Used

Monthly EMI is calculated using the standard EMI formula:

EMI = P × R × (1 + R)^N / ((1 + R)^N − 1)

Where:

- P = Loan Amount
- R = Monthly Interest Rate
- N = Total Number of Monthly Payments
